Tabit — это легкий браузер, разработанный на C++ с использованием Qt 6. Он предназначен для быстрого просмотра веб-страниц с минималистичным интерфейсом.

Для установки браузера **Tabit** выполните следующие шаги:

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/render69/Tabit.git
   ```

2. Перейдите в директорию проекта:
   ```bash
   cd Tabit
   ```

3. Соберите проект с помощью CMake:
   ```bash
   mkdir build
   cd build
   cmake ..
   make
   ```

4. Запустите браузер:
   ```bash
   ./Tabit
   ```

Убедитесь, что у вас установлены зависимости для работы с Qt 6.
