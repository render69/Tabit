## üì• Installation  

### **Clone the repository**  
```bash
git clone https://github.com/render69/Tabit.git
```

### **Navigate to the project directory**  
```bash
cd Tabit
```

### **Build the project using CMake**  

1. Create a `build` directory:  
    ```bash
    mkdir build
    cd build
    ```

2. Generate build files:  
    ```bash
    cmake ..
    ```

3. Compile the project:  
    ```bash
    make
    ```

### **Run the browser**  
```bash
./Tabit
```
